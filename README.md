A high-performance, efficient, cross-platform 2D game engine solution.

## 鸣谢 | Contributions

+ [花明的个人主页](https://github.com/naikekongwo)
+ [烛常的个人主页](https://github.com/shalltyzhuchang)
+ [罔极的个人主页](https://github.com/brid-ht)

在OpenCore引擎的初期开发中，**花明**设计并实现了场景系统、最基本的控件系统、图形渲染系统以及动画系统的蓝本。

In the early development of the OpenCore engine, **Huaming** designed and implemented the scene system, basic control system, graphics rendering system, and the blueprint of the animation system.

**烛常**则对资源加载模块、实体相关逻辑做出了卓越的贡献。其实现了资源的异步加载和各类资源的导入方法，包括JSON清单化导入。

**Zhuchang** made outstanding contributions to the resource loading module and entity-related logic, implementing asynchronous resource loading and various resource import methods, including JSON manifest-based import.

**罔极**独立开发出了沿用至今的音频模块，同时也负责了各动画派生类的设计实现，其对引擎的柏林噪声、正弦波生成机制也做了关键的工作。

**Brid-ht** independently developed the audio module that is still in use today, and was also responsible for the design and implementation of various animation derived classes. He also made key contributions to the engine's Perlin noise and sine wave generation mechanisms.
